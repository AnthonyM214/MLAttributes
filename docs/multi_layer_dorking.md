# Multi-Layer Dorking

Use layered retrieval instead of one-shot search.

## Layer 1: Official

Search for the official site and same-domain pages first.

## Layer 2: Corroboration

Search for independent confirmation from government, registry, Google Places, OSM, and structured site metadata.

## Layer 3: Freshness

Search for current, updated, open-now, hours, and contact signals so stale pages do not outrank live proof.

## Layer 4: Fallback

Search the loose query only when stronger layers did not produce usable evidence.

## Why Layers Matter

- Prevents weak generic searches from outranking direct evidence.
- Lets the resolver keep only the best proof.
- Makes it easier to measure which layer actually helped.
